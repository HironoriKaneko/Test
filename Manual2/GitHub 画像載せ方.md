GitHub 画像載せ方

エクスプローラー内に画像を入れるためのフォルダーを作成する
(今回の場合Picフォルダー)

その中に画像を入れる
(今回はcat.jpg)
![Test](Pic/プッシュ、コミット仕方画像載せ方1.png)

README.mdファイルをTeraPadで開く

画像を載せるためのコードを書く
(今回の場合`![Test](Pic/プッシュ、コミット仕方画像載せ方1.png)`)

SourceTreeでプッシュ、コミット後
GitHubを見ると画像が表示されている。
![Test](Pic/プッシュ、コミット仕方画像載せ方2.png)
